# コミュニティ

A2UI コミュニティへようこそ。私たちは一緒に、エージェント主導 UI の未来を作っています。

## 参加する

A2UI は Apache 2.0 ライセンスのオープンソースプロジェクトです。開発者、研究者、そして agentic UI の発展に関心のある人を歓迎します。

## コミュニティ紹介

A2UI で何か作りましたか。ぜひ共有してください。

成果を共有する最善の方法は、**[GitHub Discussions](https://github.com/a2ui-project/a2ui/discussions)** に短い説明、コードへのリンク、4 分以内のデモ動画を投稿することです。注目すべきプロジェクトは、ドキュメントやリリースノートで紹介します。

**コミュニティ製レンダラー** は [エコシステムのレンダラー一覧](renderers.md) ページに一覧があります。コミュニティ実装や自分の実装を追加する方法を確認してください。

**公式サンプル** は [リポジトリ内](../../../samples/README.md) にあり、良い A2UI プロジェクトの参考になります。

## プロジェクトパートナー

A2UI は Google Opal、Flutter（GenUI SDK）、Gemini Enterprise、AG-UI / CopilotKit、A2A チームと協力して開発されています。詳しくは [A2UI in the World](a2ui-in-the-world.md) を参照してください。

大きな貢献をしている方は、ぜひこの一覧に名前を追加してください。

## 行動規範

私たちは、誰もが歓迎される包摂的な環境を提供することに取り組んでいます。参加者には次が求められます。

- 敬意と配慮を持つ
- 新規参加者を歓迎し、手助けする
- コミュニティ全体にとって最善のことを考える
- 他者への共感を示す

不適切な行動は許容されません。懸念があればプロジェクトのメンテナーに報告してください。

## 表彰

すべての貢献に感謝します。貢献者は次の場所で認識されます。

- GitHub の contributor graph
- リリースノート
- プロジェクトの謝辞

大きな貢献をした方は、メンテナーチームに招待されることがあります。

## 最新情報を受け取る

- 通知を受け取るために GitHub リポジトリを Watch する
- ブックマークと応援のために Star を付ける
- 新しいバージョンを受け取るために Releases をフォローする

## 貢献方法

**[github.com/a2ui-project/a2ui](https://github.com/a2ui-project/a2ui)**

- **Issue を報告する**: バグを見つけたら [Issue を開く](https://github.com/a2ui-project/a2ui/issues)
- **レンダラーを作る**: 計画中のフレームワークは [ロードマップ](../roadmap.md) を確認する
- **例を共有する**: X/LinkedIn で `#A2UI` を付けて投稿するか、[ディスカッション](https://github.com/a2ui-project/a2ui/discussions) を始める
- **ドキュメントを改善する**: `docs/` ディレクトリへの PR を歓迎する

## 質問はありますか？

- [ドキュメント](../introduction/what-is-a2ui.md) を確認する
- [GitHub Discussions](https://github.com/a2ui-project/a2ui/discussions) を検索する
- [GitHub Issues](https://github.com/a2ui-project/a2ui/issues) で質問する

A2UI コミュニティの一員になってくれてありがとうございます。
