# A2UI Composer

CopilotKitによる、エージェントのための完全に機能するインタラクティブなウィジェットを作成し、プレビューするためのビジュアルウィジェットビルダーです。

[![A2UI Composer](assets/A2UI-widget-builder.png)](https://go.copilotkit.ai/A2UI-widget-builder)

[A2UIウィジェットビルダーを試す](https://go.copilotkit.ai/A2UI-widget-builder)

## 特徴
- リアルタイムプレビュー
- 生成AI主導の作成
- 即時のコード生成
- CopilotKitとの統合
- Angular、Flutter、Litのサポート
