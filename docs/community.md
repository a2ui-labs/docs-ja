# コミュニティ

A2UIコミュニティへようこそ！私たちはエージェント主導のインターフェースの未来を共に築いています。

## 参加する方法

A2UIは Apache 2.0 ライセンスの下で公開されているオープンソースプロジェクトです。開発者、研究者、そしてエージェントベースのユーザーインターフェース (Agentic UI) の発展に関心のあるすべての人の貢献を歓迎します。

## プロジェクトパートナー

A2UIは、多くの組織とのコラボレーションを通じて開発されています：

### Google Opal

[Opal](http://opal.google)は、ユーザーが自然語を使用してAIミニアプリを構築、編集、および共有することを可能にします。Opalチームは、A2UI開発の主要な貢献者でした。

### Flutter

[Flutter向けGenUI SDK](https://docs.flutter.dev/ai/genui)は、Flutterアプリケーションで動的でパーソナライズされたUIを生成するためのUI宣言形式としてA2UIを使用しています。

### Gemini Enterprise

A2UIは、[Gemini Enterprise](https://cloud.google.com/gemini-enterprise?e=48754805)に統合されており、エンタープライズアプリケーション内でカスタムエージェントがリッチでインタラクティブなUIをレンダリングできるようにしています。

### AG UI / CopilotKit

[AG UI](https://ag-ui.com/)と[CopilotKit](https://www.copilotkit.ai/)は、A2UIとの即時の互換性を提供し、開発者がエージェントから動的UIをレンダリングし、状態が同期されたリッチなアプリケーションを構築できるようにします。

### A2A

Googleの[A2Aチーム](https://a2a-protocol.org/)は、A2A技術運営委員会 (TSC) の支援とともに、A2UI開発の主要な貢献者でした。

### ... その他多数

多くの組織や個人がA2UIの発展に貢献しています。

A2UIに重大な貢献をされた場合は、こちらに名前を追加してください。

## 行動規範

私たちは、すべての人にとって歓迎され、包摂的な環境を提供することに尽力しています。すべての参加者は、以下を遵守することが期待されます：

- 敬意を払い、配慮のある態度を持つこと
- 初心者を歓迎し、サポートすること
- コミュニティにとって最善なことに集中すること
- 他の人に共感すること

不適切な行動は容認されません。懸念事項がある場合は、プロジェクト管理者に報告してください。

## 貢献に対する表彰

すべての貢献に感謝します！貢献者は、以下の項目で表彰されます：

- GitHubの貢献者グラフ
- リリースノート
- プロジェクトの謝辞

重大な貢献者は、プロジェクトの管理者 (Maintainer) チームに招待される場合があります。

## 最新情報を入手する

- 通知を受け取るには、**GitHubリポジトリをWatch**してください。
- ブックマークし、サポートを表明するには、**リポジトリにStar**を付けてください。
- 新しいバージョンに関する通知を受け取るには、**リリースをフォロー**してください。
